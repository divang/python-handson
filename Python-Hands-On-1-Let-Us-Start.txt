Python Quickstart
Python is an interpreted programming language, this means that as a developer you write Python (.py) files in a text editor and then put those files into the python interpreter to be executed.

1: Create a first python code file:
> Open Text Editor and create a file: helloworld.py
>> print("Hello, World!")
python helloworld.py

2: Run python command via command prompt:
> python
print("Hello, World!")
exit()

3: Python Indentation
if 5 > 2:
   print("Five is greater than two!")

4: Python Indentation- Syntax Error:
if 5 > 2:
  print("Five is greater than two!")
         print("Five is greater than two!")

5: Python Variables
> x = 5
> y = "Hello, World!"

6: Comments- Single line
#This is a comment.
print("Hello, World!")

7: Commnets- Multiple lines
"""
This is a comment
written in
more than just one line
"""
print("Hello, World!")

8: Creating Variables
x = 5
y = "John"
print(x)
print(y)

x = 4       # x is of type int
x = "Sally" # x is now of type str
print(x)

9: Casting
x = str(3)    # x will be '3'
y = int(3)    # y will be 3
z = float(3)  # z will be 3.0

10: Get the Type
x = 5
y = "John"
print(type(x))
print(type(y))

11: Single or Double Quotes?
x = "John"
# is the same as
x = 'John'

12: Case-Sensitive
